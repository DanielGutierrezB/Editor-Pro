Motion-Pro

Generador automático de motion graphics para clases educativas — Plugin para Adobe Premiere Pro

¿Qué es Motion-Pro?

Motion-Pro es una herramienta dentro de Editor-Pro que analiza la transcripción de una clase y genera automáticamente motion graphics animados usando Remotion (React). Los clips se renderizan como MP4 y se colocan directamente en la línea de tiempo de Premiere.

Los 3 Pasos

Paso 1 Análisis

La IA analiza la transcripción completa (con timestamps) y propone momentos donde motion graphics agregarían valor educativo. Cada propuesta tiene: tipo de animación, tiempos exactos, descripción y prioridad.

Prompt editable: "Editar Prompt" dentro del paso 1 controla cómo la IA decide qué momentos proponer.

Paso 2 Selección y Generación

Revisa las propuestas, filtra por tipo, selecciona cuáles generar. Al generar, cada motion pasa por: LLM genera TSX → Remotion renderiza MP4 → se coloca en la timeline de Premiere. Los clips se van agregando conforme terminan.

Paso 3 Panel de Control

Cada motion generado aparece como una ficha con: versión (dropdown), link a Remotion Studio, feedback con imágenes de referencia, regenerar. Cada nueva versión se apila encima en la timeline.

8 Tipos de Animación

comparison
Dos cards lado a lado comparando A vs B. Ideal para ventajas/desventajas, antes/después.
steps
Un paso a la vez con progress dots. Para procesos secuenciales, tutoriales.
icons
2-4 íconos SVG con labels. Para categorías, listados de conceptos.
chart
Barras o líneas animadas. Para datos, estadísticas, comparativas numéricas.
title
Ícono centrado + título + subtítulo. Para intros de sección.
cards
Tarjetas horizontales con flechas. Para flujos causa-efecto.
diagram
Cajas conectadas con líneas. Para arquitectura, flujos de datos.
ui
Mockup de interfaz centrado. Para demos de formularios, navegación.

Prompts de Generación

La generación usa 3 documentos que se envían completos al LLM como system prompt, más un template de código e instrucciones por tipo:

DocumentoContenidoEditable
SYSTEM_PROMPT.md Reglas maestras: componentes (Safe, E, Fd), paleta de colores, tipografía, tipos de animación, anti-patterns, safe zones Sí — desde "Prompts de Generación" en el panel
STYLE_GUIDE.md Tamaños para 1080p, spacing entre elementos, reglas de charts (barras, líneas, donut), reglas de UI mockups Sí
DESIGN_FUNDAMENTALS.md 12 Principios Disney aplicados, Gestalt, teoría del color, tipografía, composición, timing, dinamismo visual Sí
Template TSX Código base que el LLM debe usar: imports, paleta C, componentes Safe/E/Fd inline En prompts.js
Instrucciones por Tipo Reglas específicas para cada uno de los 8 tipos (tamaños, layout, comportamiento) Solo lectura en UI

Feedback y Versionado

Cada motion puede tener múltiples versiones. Al enviar feedback:

  1. El LLM recibe: el TSX actual + tu feedback + los 3 docs de diseño + instrucciones del tipo
  2. Genera una nueva versión del TSX
  3. Remotion la renderiza a MP4
  4. Se coloca en el track superior de la timeline
  5. El dropdown de versión se actualiza
  6. Todo el feedback se registra en FEEDBACK.md junto al proyecto

Remotion Studio

Desde el panel de control, el botón "Remotion" abre Remotion Studio en el browser. Ahí puedes:

  • Ver el preview en tiempo real de cualquier composición
  • Editar el TSX directamente y ver cambios al instante
  • Renderizar manualmente desde el browser

Comando manual: cd motion-render && npx remotion studio

Servidor Local
